Source code documentation
=========================

The source code of *Nemoh2HDB* is composed of 9 Python scripts:
 - HDB5.py which stores and post-processes the data resulting from *Nemoh*;
 - bem_reader.py for reading the input and output files of *Nemoh*;
 - discretization_db.py for fixing the discretization parameters;
 - hydro_db.py for dealing with hydrodynamic database;
 - environment_db.py for defining the environmental data;
 - hydrostatic_db.py for storing hydrostatic stiffness matrices;
 - wave_drift_db.py for storing wave drift polar coefficients;
 - body_db.py for writing the body data into the *.hdb5* output file;
 - wave_dispersion_relation.py for computing the wave dispersion relation.
